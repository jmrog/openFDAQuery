Welcome

The purpose of this page is to simplify the task of making queries to the openFDA data sets. Simply use the form below to make your query; you do not need to know the details of the openFDA API.

NOTE: Currently only the most basic of searches are supported. We hope to add additional capabilities soon.

Choose a data set:









What do you want to search for in drugs? Searches can consist of a field and a term, but the field is optional (a search with no field searches all fields). Spaces are allowed in the term only. To search for multiple items (i.e., multiple field/term pairs, where the field is optional), use the "Add Search Item" button.

Not sure what fields are available for searching through? See the lists of fields provided on the openFDA website for the data you're interested in; for example, here is the reference for adverse events for devices: https://open.fda.gov/device/event/reference/#field-by-field-reference.
